# GuideScope 再起動・再開手順

## 開発サーバー再起動（Vite）
1. 既存の dev サーバーを停止します（ターミナルで `Ctrl + C`）。
2. プロジェクトへ移動します。
   - `cd /Users/masayuki/Dev/guidescope`
3. 依存関係が更新されている場合のみ再インストールします。
   - `pnpm install`
4. 開発サーバーを起動します。
   - `pnpm dev`
5. ブラウザで表示を確認します。
   - `http://localhost:5173`

## パッケージ再ビルド（必要な場合）
コアパッケージの反映が必要な場合のみ実行します。

- `pnpm build:packages`

## 本番ビルド確認
配布用の静的ビルドが必要な場合に実行します。

- `pnpm build`
- `pnpm preview`

## 変更点の簡易確認ポイント
- スタンダード: 冒頭に「サマリー」「引用文献」が出力される
- プロ: 従来の出力に「サマリー」が冒頭で追加される
- スタンダード時にオプション系トグルが操作不可になっている
